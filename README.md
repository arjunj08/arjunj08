## 🚀 Main Projects

### 1️⃣ **SkillBridge** — AI EdTech Platform

*Bridging the gap between college coding and industry-ready skills*

**Problem:** Indian CS students ace exams but struggle with practical placement prep.  
**Solution:** AI-powered platform with mock interviews, skill gap analysis, and job readiness scoring.

**What's Live:**
- 🔐 Role-based authentication & secure login
- 📚 Problem bank with company tags & AI filtering
- 🤖 AI-powered mock interview simulator
- 📊 Job Readiness Score with skill gap analysis
- 👥 Group Study Rooms for peer learning
- 🏆 Leaderboard system
- 📜 Certification tracking
- 👨‍🏫 Professor Portal

**Tech Stack:**
```
Frontend:   React 18 + Vite + Tailwind CSS
Backend:    Node.js + Express.js
Database:   MongoDB
Real-time:  Socket.io
Auth:       JWT
Integrations: Google Sheets API + Claude AI
```

**Current Status:** Building v2 with enhanced AI features | Target: Production Q3 2026

### 2️⃣ **AUISC EventSync** — Club Management Platform

*Full-stack system managing university student club operations*

**Features:**
- 📋 Kanban task management with real-time sync
- 📅 Meeting scheduling & management
- 🤖 AI-powered meeting summaries
- 📹 Integrated video calling
- 📸 Media galleries & file storage
- 🔔 Real-time notifications via Socket.io
- 📱 React Native mobile app (Android APK)

**Tech Stack:**
```
Web:   React + Node.js + MongoDB + Socket.io
Mobile: React Native + Expo + EAS Build
Auth:   JWT + Role-based access
Video:  Agora SDK
```

**Current Status:** In production for 500+ student club members
